## Instalação e rodando o projeto
```sh
$ git clone https://github.com/andersonribeirodemoraes/conceitos-reactjs.git
$ cd conceitos-reactjs  
$ yarn
$ yarn start

Rodando em: http://localhost:3000
```

## Rodando os testes
```sh
$ yarn test
```
